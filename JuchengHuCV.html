Reinforce Learning AI Championship 1st Place

Tic-Tac-Toe Player Agent Design  2019

In the competition held by KCL I was supposed to train a tic-tac-toe player agent through reinforcement learning and all agents from all competitors would be put together to determine the champion.

 

I managed to write a program which can automatically try a series of random parameters such as the one used to balance the exploitation and exploration in Epsilon-Greedy algorithm and return the marks through ELO method. By analysing hundreds of random games generated by my helper program, I found that a winning move always tend to gain a higher pieces density in a local area.

 

Base on this finding, I created my reward function and its performance won me the first place.
